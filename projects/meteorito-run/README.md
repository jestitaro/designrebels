# Meteorito Run · Dino Cup

Minijuego arcade independiente de Dino Cup.

## Controles

- Espacio o flecha arriba: saltar.
- Clic o toque sobre el juego: saltar.
- P o Escape: pausar.
- Enter: iniciar o volver a jugar.

Incluye puntaje, niveles, récord local, sonidos opcionales y diseño responsive para desktop y mobile.
